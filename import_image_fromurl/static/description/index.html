Import Product, Customer, Supplier Images From Direct URL / CSV / XLS File


This module allow you to import images directly from url / csv file / excel file contains mapped field and url.
For CSV/XLS file you have an option to select mapping field so that you can add mapping filed and url.
For single object like product,customer,supplier you can manually add or modify image url

Key features:
  • Easy To Use.
  • Import images from url for product, customer, supplier.
  • Import Via directly adding url, csv file, xls file.
  • For csv / xls file you have an option to select mapping field like database id, name, internal reference.
  • Supporting Create and Write option. So if you update URL and Edit and Save it read again and set Image.
  • Maintains Log, so user can check when file was imported and check error if any.
  • Set access right, so specific user will set image though url/csv/xls file.
To enable import functionality, you have to select Upload Images From URL/File access from setting --> users --> users --> Assess Right --> Other
If you set access right IMAGE URL field will visible in product form view.
If you set access right IMAGE URL field will visible in Partner(Customer/Supplier) form view.
Goto Sales --> Configuration --> Upload Bulk Images --> Upload Image URL File
In Applied Object drop-down you have three options 1.Supplier 2.Customer 3.Product, suppose user select Product.
Now user have four options in Select Map Filed 1.Database ID 2.Product Name 3.Product Default Code 4.Product EAN13 Barcode, suppose user select Database Id.
From Select File Type user have two options 1. CSV file 2.XLS file.
From Applied Object drop-down user select Customer.
Now user have two options in Select Map Filed 1.Database ID 2.Partner Name, same fields are available for Supplier too.
From Select File Type user have two options 1. CSV file 2.XLS file.
